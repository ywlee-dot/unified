Unified Workspace와 n8n Cloud를 연결하는 방법을 안내합니다.

!
전체 구조 이해

어떤 n8n 워크플로우든 앞뒤에 두 개 노드만 붙이면 연동됩니다.

흐름 요약
프론트에서 버튼 클릭 → 백엔드가 Webhook 호출 → n8n이 작업 실행 → 완료 후 콜백으로 결과 전달 → 프론트에 표시
1
Webhook 노드 설정 (입구)

n8n 워크플로우의 맨 처음에 추가합니다.

노드 추가 방법:

캔버스에서 + 클릭 → Webhook 검색 → 추가

Parameters 탭
HTTP Method POST
Path 프로젝트에 맞게 설정 (예: summarize)
Authentication None
Respond Immediately
Respond 설정 주의
반드시 Immediately로 설정해야 합니다. 다른 옵션을 선택하면 에러가 발생합니다.
2
작업 노드 구성 (중간)

Webhook과 콜백 사이에 원하는 작업을 자유롭게 연결합니다.

주의사항:

마지막 작업 노드의 출력에 tracking_id가 포함되어야 합니다. 콜백에서 이 값으로 실행 기록을 매칭합니다.

+ 클릭 → Code 검색 → 추가
Language: JavaScript / Mode: Run Once for All Items

// Webhook에서 받은 데이터 꺼내기 const webhook = $('Webhook 노드 이름').first().json; const body = webhook.body || webhook; // === 여기에 실제 작업 로직 작성 === const result = { message: "작업 완료" }; // ================================= // tracking_id를 반드시 포함해서 반환 return [{ json: { ...result, tracking_id: body.tracking_id || "" } }];
노드 이름 확인
$('Webhook 노드 이름')에서 이름은 n8n 캔버스에 표시된 노드 이름과 정확히 일치해야 합니다.

중간 노드가 많아도 마지막 노드에서 Webhook의 tracking_id를 가져오면 됩니다.

// 마지막 작업 노드에서 const tracking_id = $('Webhook 노드 이름').first().json.body.tracking_id; return [{ json: { ...기존결과, tracking_id } }];
3
HTTP Request 콜백 노드 설정 (출구)

워크플로우의 맨 마지막에 추가합니다. 작업 결과를 백엔드에 전달하는 역할입니다.

노드 추가 방법:

+ 클릭 → HTTP Request 검색 → 추가

기본 설정
Method POST
URL https://unified-backend-7nja.onrender.com/api/webhooks/n8n/callback
Headers 설정
Send Headers ON
Specify Headers Using Fields Below
Name Value
X-N8N-Callback-Secret tucon-n8n
Body 설정
Send Body ON
Body Content Type JSON
Specify Body Using JSON

JSON Body 내용:

{ "execution_id": "{{ $json.tracking_id }}", "status": "completed", "result_data": {{ JSON.stringify($json) }}, "error_message": null, "finished_at": "{{ new Date().toISOString() }}" }
Secret 값 확인
X-N8N-Callback-Secret 값은 백엔드 환경변수 N8N_CALLBACK_SECRET과 정확히 동일해야 합니다.
4
저장 및 Publish

모든 노드를 연결한 후:

  1. Ctrl + S 또는 메뉴에서 저장
  2. 우측 상단 Publish 버튼 클릭
Publish 필수
Publish하지 않으면 프로덕션 webhook URL이 동작하지 않습니다. 수정할 때마다 다시 Publish 해야 합니다.
✓
최종 체크리스트
  • Webhook 노드의 Respond가 Immediately로 설정되어 있다
  • 콜백 URL이 https://unified-backend-7nja.onrender.com/api/webhooks/n8n/callback 이다
  • 헤더 X-N8N-Callback-Secret 값이 백엔드와 동일하다
  • JSON Body에 tracking_id가 포함되어 있다
  • 워크플로우가 Publish 되어 있다
